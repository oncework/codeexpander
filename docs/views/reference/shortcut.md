# 常用快捷键

| Command      | Keybinding | Source  | When           |
| :----------- | :--------- | :------ | :------------- |
| 创建分组     | ⌘ + T      | Default | software focus |
| 创建片段     | ⌘ + I      | Default | Software focus |
| 快速创建片段 | ⌥ + \]     | Custom  | Global         |
| 搜索工具栏   | ⌥ + Space  | Custom  | Global         |

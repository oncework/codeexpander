## 环境配置

### Windows OS

您可按自己的需求配置相关的脚本运行环境即可。

> 如你需要运行 Python、JavaScript、NodeJs 脚本等。

```text
$ node -v
$ sh
$ python -v
```

如果缺少其中一些命令，请自行下载并安装。

### Mac OS
